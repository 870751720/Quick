# 叙事前提与第一章体验

## 文档定位

- 用途：定义当前有效的世界观基调、玩家身份，以及第一章从种田生活转入冒险世界的体验结构。
- 适用范围：开场村庄、主角背景、地主关系、种田阶段与战士路线的叙事衔接。
- 维护状态：草案。
- 边界：种植、经济、战斗等系统的完整规则应分别记录在独立系统文档中；本文只定义跨系统体验目标。

## 当前有效方案

### 世界与主角

游戏采用黑暗奇幻背景，但黑暗主要来自人物、制度与故事，而非持续阴暗的环境。初始村庄表面正常、明亮且富有生活气息；压迫隐藏在土地、债务、交易和村民习以为常的规则中，需要玩家观察和思考才能识别。

玩家由现代世界魂穿到一名落魄年轻人身上。年轻人没有土地与资本，受制于当地地主。若不主动改变人生路径，他可以通过劳动改善生活，却会始终被锁在地主控制的生产关系中。

### 第一阶段：好玩的种田生活

前期以种田作为主要玩法。基础循环为：

1. 购买能够快速生长的作物种子。
2. 播种并浇水。
3. 等待短周期成熟并收割。
4. 售卖作物，继续购买种子和能力升级。

种田必须具有真实、持续的正反馈，不能仅作为故意折磨玩家的假玩法。成长方向包括：

- 扩大格子背包容量；
- 提升播种、浇水与收割效率；
- 降低种子购买价格；
- 提高作物售卖价格；
- 通过收割副产物强化农具。

如果玩家只专注种田，应当认为自己正在玩一款完成度不错的种田游戏，而不是被系统明显惩罚。

### 舒适牢笼

种田成长提供的是局部效率提升，而不是身份自由。地主应通过土地所有权、种子来源、收购渠道或债务规则掌握循环的关键节点，使玩家赚得更多、劳动得更高效，却仍然依附于现有秩序。

约束设计原则：

- 不采用粗暴没收全部收益或无限追加欠款的方式否定玩家努力；
- 玩家获得的升级、背包和工具强化必须永久有效；
- 困局来自收益用途和人生选择受限，而非数值作弊；
- 前期必须埋下可验证的异常与出路，让醒悟来自玩家判断；
- 种田路线可以长期继续，不强制所有玩家在固定时间转职。

### 第二阶段：成为战士

世界真正开放的条件不是达到某个农业等级，而是玩家意识到“继续优化劳动无法获得自由”，并抓住一个成为战士的契机。

成为战士后：

- 村外区域、势力、任务与战斗内容逐步开放；
- 前期种田积累仍有价值，背包、体力管理、工具成长和资源经验可迁移到冒险；
- 玩家第一次获得不依赖地主的资源来源和身份选择；
- 种田不被废弃，而是从唯一生路变成玩家可以自主选择的经营能力。

### 战斗觉醒契机：商店中的遗失道具

玩家频繁交互的商店老板持有一件从年轻人身边捡到的特殊道具。老板知道它来自玩家，但不知道用途，也没有主动归还。该道具在商店场景中应长期可见，使玩家能够逐渐注意和记住它；获得道具后，玩家原身体的战斗能力被激活，战斗系统与村外世界随之开放。

道具提供三条有效获取路径：

1. **老板自愿归还**：玩家从未购买“降低种子价格”或“提高作物售价”的议价成长，并通过持续原价交易让老板累计获得固定利润后触发。老板将道具作为对长期优质顾客的回报。判定基于老板实际获得的累计利润，而非玩家消费总额。
2. **高价购买**：玩家可以直接买下道具，但价格极其高昂，构成种田路线的长期显性目标。价格必须在纯种田玩法中真实可达，不得只是伪选项。
3. **偷盗取得**：玩家在老板未注意时窃取道具。偷盗会改变老板关系、村庄评价及后续交易条件，不能成为无代价的最优解。

三条路径分别表达信任、积累和反抗，不设置唯一正确选择。道具一旦取得，战斗系统永久开启；前期农业成长继续有效。

### 偷盗能力的自然领悟

游戏初期不直接提供偷盗技能。玩家在 NPC 未注意时成功拿走三件不属于自己的物品后，自然领悟偷盗。前三次行为不是无反馈的隐藏计数：每次成功都应通过角色动作、短暂心跳声、NPC 视线变化或内心文字，让玩家察觉该行为正在形成一种能力。

规则约束：

- 只有明确属于 NPC 或场所、且玩家知道并非免费拾取的物品才计数；
- 必须在物主或见证者未发现的情况下完成，失败行为不计入领悟进度；
- 三次目标应分布在玩家自然会探索的位置，避免要求无提示地点击随机装饰；
- 领悟前允许拿取低价值物，但商店核心道具需要正式偷盗能力；
- 偷盗判定、察觉、赃物和后果需要在独立系统文档中定义，不能只作为一次剧情机关；
- 不在 UI 中显示“偷盗 1/3”进度，以保留玩家自行发现规则的体验。

## 核心体验目标

第一章需要制造两次都成立的认知：

1. “这是一个反馈良好、可以不断经营优化的种田游戏。”
2. “我变得越来越擅长劳动，但我并没有因此拥有自己的人生。”

黑暗感应在第二层认知形成时出现。游戏不替玩家宣讲结论，而是让经济结构、NPC 态度和可探索线索共同证明它。

## 待定事项

- 主角与原身体的姓名、家庭关系及债务来源；
- 快速生长作物的名称、生态代价与用途；
- 地主掌控生产闭环的具体规则；
- 遗失道具的名称、来源、外观与激活战斗能力的原因；
- 老板自愿归还所需的累计利润和高价购买价格；
- 偷盗系统的察觉、后果、赃物及关系规则；
- 留在种田路线的长期结局与内容边界；
- 种植系统和村庄经济系统的独立专题文档。
