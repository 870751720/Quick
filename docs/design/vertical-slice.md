# 《余烬荒原》首个可玩切片

状态：生效中（原型）；更新：2026-07-11。

原创 2D 俯视角动作 RPG。战斗目标是移动清怪、连续攻击和短冲刺带来的直接爽感，不采用魂系精力管理与强动作承诺。后续围绕技能、装备和敌人组合扩展。

WASD 移动、鼠标朝向、左键/空格连续攻击、右键/Shift 短冲刺、R 重试。攻击不消耗精力，伤害 28；短冲刺不消耗资源。守卫生命 150、伤害 18；玩家受击后获得短暂无敌，降低连续硬直挫败。

玩家、敌人和环境统一使用 Puny Characters 像素资源包；按 32×32 帧与 8 方向解析待机、移动和剑击动画。HUD 常驻。攻击、命中、冲刺、受击、胜利和失败使用 Web Audio 原创程序音效，首次点击开始按钮后解锁。粒子、受击/死亡动画和音量/静音开关仍待补齐。

战斗改为连续波次：第 1 波 3 只怪物，之后逐波增加，最多同时 8 只；怪物生命随波次小幅提高。清空后延迟生成下一波，形成持续战斗循环。

从第 2 波开始，每个偶数波固定出现精英“赤牙督军”。精英体型 1.28 倍、生命 2.4 倍、伤害 1.6 倍、移速略低；金币 4 倍、经验 3 倍且必掉回血药。精英头顶持续显示名字和金色血条。

所有角色脚下绘制半透明椭圆阴影。玩家命中显示浮动伤害数字，敌人短暂闪白、抖动并沿攻击方向后退；玩家受击显示红色伤害数字并闪烁。跳字约持续 0.75 秒并向上飘移，不参与战斗判定。

每只怪物死亡必掉 5–12 金币，38% 概率额外掉落回血药。玩家靠近 55 像素自动拾取；回血药进入背包而不直接生效，按 Q 消耗一个并恢复 25 生命，满血时不会消耗。掉落时金币或绿色碎屑径向爆散并受重力影响。金币与背包在本局累计，死亡或按 R 后重置。

玩家从 1 级开始，升级所需经验为 `20 + 等级 × 10`；击杀经验为 `10 + 当前波次 × 2`，因此第一波即可升到 2 级。每级最大生命 +10、攻击 +4，升级立即回满生命并触发金色爆发与升级音效。当前成长仅限单局，不保存。

人工验证玩家能否理解攻击、冲刺、自动拾取与波次目标；爆落是否清晰且不过度遮挡战斗；补给概率是否能支撑至少 3–5 波。记录最高波次、金币和死亡原因，最终体验验收由用户完成。
