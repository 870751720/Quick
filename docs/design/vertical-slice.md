# 《余烬荒原》首个可玩切片

状态：生效中（原型）；更新：2026-07-11。

原创 2D 俯视角高风险动作 RPG。借鉴“观察前摇—管理精力—承诺动作—承担失误”的节奏，不复制既有作品内容。目标是在 1–3 分钟内理解距离、精力和出手时机的取舍并击败守墓人。

WASD 移动、鼠标朝向、左键/空格攻击、右键/Shift 翻滚、R 重试。玩家生命/精力 100；攻击消耗 24、伤害 34；翻滚消耗 30，持续 0.42 秒，前 0.28 秒无敌。守墓人生命 150、伤害 28，前摇 0.46 秒。

敌人暖色高亮表达前摇，姿态表达攻击阶段，HUD 常驻。当前使用原创代码几何占位，无外部资产。音频、粒子、手柄震动、可访问性开关尚缺失，不视为最终品质。

人工验证玩家能否理解精力与前摇，死亡后能否判断原因，预期 2–6 次尝试通关。记录通关时间与死亡次数，最终体验验收由用户完成。
